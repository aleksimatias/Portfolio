DIGIREACTOR

DigiReactor was a school project that was done in co-operation with
TUAS, University of Turku and Turku Business Region.

The application was done by a student group using React Native.
I created the initial UI mockups, and during the development phase
I was tasked to creating different page and menu views.

FRIBAKELI

Fribakeli was a school project done in a group of 3.
The application combined local weather data with coordinates of
disc golf tracks, to let users find viable tracks to play on.

The application was created using React.
I worked on the weather data, map styling and linking data together.

Fribakeli is currently a WIP project, and it may be finished this year.
